Question Paper Builder
An AI-inspired modern Question Paper Builder built using React + Vite.
This project was created to simplify the traditional question paper creation workflow used in many schools and institutions.

Inspiration:
Growing up with parents who are teachers, I personally saw how time-consuming the process of making question papers can be:

Writing questions manually
Sending them to the office
Re-typing everything in DOCX format
Fixing formatting mistakes repeatedly
Managing sections and marks manually

So I decided to build a digital solution that simplifies this workflow.

This project is currently a standalone prototype and will later become a small module inside a much bigger education-focused platform that I plan to build in the future.

Features:
Modern responsive UI
Multiple sections support
Dynamic question management
Marks distribution handling
Question preview system
Login UI
Real-time editing
Fast React-based interface
